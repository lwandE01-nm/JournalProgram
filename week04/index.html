Foundation Program Design Document

This document contains the design for the Week 04 Foundation Programs: Abstraction with YouTube Videos and Encapsulation with Online Ordering.

Program 1: Abstraction with YouTube Videos

What the Program Does

The program models a YouTube-style system that stores videos and comments. Users can view video information and comments associated with each video.

Classes and Responsibilities

Class Responsibilities
Video Store video information and comments.
Comment Store commenter name and comment text.
VideoLibrary Store and display multiple videos.

Attributes and Methods

Video

  • Attributes: title, author, length, comments
  • Methods: addComment(), getNumberOfComments(), displayVideoInfo()

Comment

  • Attributes: userName, text
  • Methods: displayComment()

VideoLibrary

  • Attributes: videos
  • Methods: addVideo(), displayAllVideos()

Class Diagram

VideoLibrary → Video → Comment

Program Flow

Create Videos → Add Comments → Store in Video Library → Display Videos and Comments

Program 2: Encapsulation with Online Ordering

What the Program Does

The program simulates an online ordering system that manages customers, addresses, products, and orders.

Classes and Responsibilities

Class Responsibilities
Product Store product information and calculate cost.
Customer Store customer information.
Address Store customer address information.
Order Manage products, shipping, labels, and totals.

Attributes and Methods

Product

  • Attributes: name, productId, price, quantity
  • Methods: calculateTotalPrice()

Customer

  • Attributes: name
  • Methods: displayCustomerInfo()

Address

  • Attributes: street, city, province, country
  • Methods: displayAddress()

Order

  • Attributes: products, customer, address
  • Methods: addProduct(), calculateTotal(), getPackingLabel(), getShippingLabel()

Class Diagram

Customer + Address → Order → Products

Program Flow

Create Customer → Create Address → Add Products → Create Order → Calculate Total → Generate Labels
